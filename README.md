# EasyEhco
EasyEhco is a simple script for configuring Ehco on Linux.

## Fetures
- [x] Supports booting on startup
- [x] Supports relay mode
- [x] Supports landing mode

## Usage
```shell
bash <(curl -Ls https://owo.nz/ehco)
```
